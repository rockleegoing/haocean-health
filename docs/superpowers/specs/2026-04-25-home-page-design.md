# 首页（便捷执法）布局完善设计方案

## 1. 概述

根据原型图，完善 Android App 首页（便捷执法）布局，添加"文书管理"快捷入口，形成两行四列的网格布局。

## 2. 现有状态

**首页当前布局（5个入口）：**
- 第一行：拍照、录音、录像、导航
- 第二行：记录

**原型图布局（6个入口）：**
- 第一行：拍照、录音、录像、导航
- 第二行：记录、文书管理

## 3. 设计方案

### 3.1 布局调整

**修改文件：** `fragment_home.xml`

**方案：保持现有第一行四列，将第二行扩展为两列**

| 位置 | 功能 | 图标 |
|------|------|------|
| (0,0) | 拍照 | `ic_camera` |
| (1,0) | 录音 | `ic_mic` |
| (2,0) | 录像 | `ic_videocam` |
| (3,0) | 导航 | `ic_navigation` |
| (0,1) | 记录 | `ic_assignment` |
| (1,1) | 文书管理 | `ic_document` (新增) |

### 3.2 新增入口

**文书管理入口：**
- 点击后跳转到 `DocumentListActivity`
- Route: `/app/document/list`
- 需确保用户已选择执法单位

### 3.3 图标资源

需要新增或确认 `ic_document` 图标资源。

## 4. 实现步骤

1. 修改 `fragment_home.xml` 布局，将第二行改为两列
2. 添加"文书管理"入口的布局
3. 在 `HomeFragment.kt` 添加点击事件处理
4. 确认或添加文档图标资源
